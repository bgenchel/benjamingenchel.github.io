Introduction

This website is a companion to the NeurIPS 2019 Machine Learning for Creativity and Design Workshop submission of the same name, which discussing a real-time convolutional neural network based musical improvisation system. Here, we provide audio, video and other content which characterizes the system and its use by performers.

Abstract

We present a simple, easy to train, and effective interactive real-time system forgenerating musical improvisations in performance with human musicians. The system comprises a Max/MSP patch communicating with a generative convolutional neural network and uses a novel data format that appears to allow improvedlearning of non-local dependencies and repetitive structure across beats within musical phrases. We have observed that this system is able to learn to generateconvincing and coherent improvisations from relatively small amounts of data, can run effectively with limited computational resources, and produces pleasing musical interactions in a live performance setting.

Audio Samples

The following audio samples showcase an improviser interacting with the system over an eight bar loop harmonized by the C7 chord. In each sample, the melody over the first four bars is played by a human improviser, while the last four are 'played' by our system — the melody is generated using the human improviser's solo as input. We use a marimba instrument to render the generated melody because in performance, these melodies are played on a marimba by our resident musical robot, Shimon.
